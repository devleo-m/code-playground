# Aula 3: Rendering e Conceitos Avançados

Bem-vindo à Aula 3! Aqui vamos abrir o capô do React.

## Estrutura da Aula

1.  [**Aula Principal**](./01-aula-principal.md): Teoria profunda sobre VDOM, Reconciliation, Ciclo de Vida e Refs.
2.  [**Aula Simplificada**](./02-aula-simplificada.md): Entenda os conceitos complexos com analogias do dia-a-dia.
3.  [**Exercícios e Reflexão**](./03-exercicios-reflexao.md): Coloque a mão na massa e desafie seu entendimento.
4.  [**Performance e Boas Práticas**](./04-performance-boas-praticas.md): Como deixar seu app React voando baixo e evitar armadilhas comuns.

## Objetivo
Ao final desta aula, você deve ser capaz de olhar para um código React e visualizar mentalmente quando e porquê ele renderiza, além de identificar gargalos de performance óbvios.
