# Aula 1 - Análise de Desempenho

Assim que você me enviar os exercícios e reflexões, avaliarei:

1. **Clareza do componente**: o JSX é legível? As props têm nomes explícitos?
2. **Responsabilidade única**: cada componente trata de um único propósito ou mistura lógica?
3. **Organização do projeto**: a estrutura criada via CLI está sendo respeitada? Os componentes ficam próximos aos estilos e exemplos?
4. **Impacto na experiência**: o componente entrega retorno visual previsível e fácil de manter?
