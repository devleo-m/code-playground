# Curso React: Componentes com CLI e Vite

Este diretório inaugura a trilha de React guiada pelo `@professor-react.mdc`. Como esta é a **Aula 1**, focamos em configurar o ambiente com ferramentas de linha de comando e em entender o que são componentes funcionais antes de avançar para renderização e hooks.

## Estrutura desta aula
- `01-aula-principal.md`: explora CLI Tools, Vite e componentes funcionais com um exemplo mínimo.
- `02-aula-simplificada.md`: reconstrói o mesmo conteúdo com analogias simples.
- `03-exercicios-reflexao.md`: exercícios para praticar e questões de reflexão sobre responsabilidade única e fluxo de dados.
- `04-performance-boas-praticas.md`: boas práticas iniciais para manter componentes leves e previsíveis.
- `05-analise-desempenho.md`: orientações para feedback futuro e perguntas para direcionar a próxima aula.

## Continuidade
Esta aula será seguida da `Aula 2`, onde introduziremos renderização condicional e depois hooks com `useState` e `useEffect`. Por enquanto, mantenha o foco em como componentes são estruturados e como as ferramentas CLI nos ajudam a começar mais rápido.
