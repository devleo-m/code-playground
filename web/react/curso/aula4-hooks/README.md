# Aula 4: Hooks - useState e useEffect

Nesta aula mergulhamos nos Hooks, a feature que modernizou o React. Aprendemos como gerenciar estado e efeitos colaterais em componentes funcionais.

## Conteúdo
1.  **Aula Principal:** História dos Hooks, `useState` em profundidade e o modelo mental do `useEffect`.
2.  **Aula Simplificada:** Analogias do Chef de Cozinha e do Robô Assistente.
3.  **Exercícios:** Cronômetro e Fetch de dados.
4.  **Performance:** Regras dos Hooks (Rules of Hooks) e Lazy Initialization.
