Links Externos

Links externos apontam para páginas de outros sites. Sempre use rel="noopener noreferrer" com target="_blank" por segurança.

Link Básico Externo

Visite o Google para fazer buscas.

Link com Título

Acesse o MDN Web Docs para documentação completa sobre web development.

Por que usar rel="noopener noreferrer"?

  • Segurança: Previne vulnerabilidade window.opener
  • Performance: noopener melhora performance
  • Privacidade: noreferrer não envia referrer
  • Boas práticas: Padrão da indústria

Exemplos de Links Externos Seguros

  • W3C
  • W3C Validator
  • GitHub

Links Internos

Links internos apontam para páginas do mesmo site. Eles abrem na mesma aba (comportamento padrão).

Exemplos de Links Internos

  • Contato
  • Produtos
  • Blog

Nota: Estes links são fictícios para demonstração. Em um site real, eles apontariam para páginas existentes.

Âncoras (Links para Seções da Mesma Página)

Âncoras permitem navegar para seções específicas da mesma página. Use o atributo id no elemento de destino e href="#id" no link.

Exemplo: Link para Seção Específica

Clique aqui para ir direto para a seção de boas práticas.

Como Funciona

  1. Defina um id no elemento de destino: 
    <section id="minha-secao">
  2. Crie um link com href="#id": 
    <a href="#minha-secao">Ir para seção</a>
  3. Ao clicar, a página rola até o elemento com aquele id

Links de Email e Telefone

Links de Email (mailto:)

Links de email abrem o aplicativo de email padrão do usuário.

Email Básico

Entre em contato: contato@exemplo.com

Email com Assunto e Corpo

Enviar Email com Assunto

Formato: mailto:email@exemplo.com?subject=Assunto&body=Corpo

Links de Telefone (tel:)

Links de telefone permitem fazer ligação diretamente (em dispositivos móveis).

Telefone Básico

Ligue para: (11) 99999-9999

Telefone com Formato Internacional

  • +55 11 99999-9999 (Brasil)
  • +1 212 555-1234 (EUA)

Dica: Use formato internacional (+55 para Brasil) para melhor compatibilidade.

Links de Download

Use o atributo download para forçar o download de um arquivo ao invés de abri-lo no navegador.

Exemplos

  • Baixar PDF
  • Baixar ZIP

Nota: O atributo download funciona apenas para arquivos do mesmo domínio (por questões de segurança).

Boas Práticas de Links

1. Texto Descritivo

❌ Ruim: Clique aqui para saber mais.

✅ Bom: Saiba mais sobre nossa empresa.

Por quê? Leitores de tela leem links fora de contexto. "Clique aqui" não fornece informação útil.

2. Links Externos Seguros

Sempre use rel="noopener noreferrer" com target="_blank": 

<a href="https://exemplo.com" 
   target="_blank" 
   rel="noopener noreferrer">
    Link Seguro
</a>

3. Atributo Title

Use title quando o texto do link é curto mas precisa contexto: 

<a href="https://exemplo.com" 
   title="Visite nosso site principal para mais informações">
    Saiba mais
</a>

4. Indicar Links Externos

Considere indicar visualmente quando um link é externo:

  • Site Externo (abre em nova aba)

5. Navegação por Teclado

Certifique-se de que todos os links sejam acessíveis por teclado. Use Tab para navegar entre links e Enter para ativá-los.

Checklist de Links

  • ✅ Texto descritivo (não "clique aqui")
  • ✅ Links externos com rel="noopener noreferrer"
  • ✅ Title quando necessário para contexto
  • ✅ Acessíveis por teclado
  • ✅ Funcionam em diferentes navegadores
  • ✅ Testados e validados
