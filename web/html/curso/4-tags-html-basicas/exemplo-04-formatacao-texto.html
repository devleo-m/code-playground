Formatação de Texto em HTML

Este documento demonstra todas as tags de formatação de texto disponíveis em HTML.

Strong vs B: Importância vs Negrito Visual

Strong (<strong>) - Importância Semântica

A tag <strong> indica que o texto tem forte importância. Visualmente, geralmente aparece em negrito, mas tem significado semântico.

Exemplo: Este texto é muito importante e precisa ser destacado para o leitor.

B (<b>) - Negrito Visual

A tag <b> torna o texto visualmente em negrito, mas sem significado semântico. Use apenas para destaque visual.

Exemplo: O nome científico é Canis lupus (apenas destaque visual, sem importância semântica).

Diferença e Quando Usar

  • <strong>: Use quando o texto for realmente importante
  • <b>: Use apenas para destaque visual sem importância
  • Prefira <strong> na maioria dos casos

Nota: Leitores de tela enfatizam mais o texto dentro de <strong>, tornando-o mais acessível.

Em vs I: Ênfase vs Itálico Visual

Em (<em>) - Ênfase Semântica

A tag <em> indica ênfase no texto. Visualmente, geralmente aparece em itálico, mas tem significado semântico.

Exemplo: Eu realmente preciso que você entenda isso. (A palavra "realmente" recebe ênfase no contexto da frase)

I (<i>) - Itálico Visual

A tag <i> torna o texto visualmente em itálico, mas sem significado semântico. Use para termos técnicos, nomes científicos, pensamentos ou palavras estrangeiras.

Exemplos:

  • Nome científico: Homo sapiens
  • Pensamento: Ele pensou: Será que isso vai funcionar?
  • Termo técnico: O protocolo HTTP é usado na web
  • Palavra estrangeira: A palavra rendezvous vem do francês

Diferença e Quando Usar

  • <em>: Use para dar ênfase no contexto
  • <i>: Use para termos técnicos, nomes científicos, pensamentos
  • Prefira <em> quando houver ênfase real

Mark (<mark>) - Texto Marcado

A tag <mark> marca ou destaca partes do texto, geralmente com fundo amarelo. É como usar um marcador de texto.

Exemplos de Uso

Este é um texto normal com uma palavra marcada para destaque.

Em resultados de busca, termos pesquisados podem aparecer marcados para facilitar a identificação.

Você pode usar mark para destacar informações relevantes que o usuário precisa notar, como avisos importantes ou termos-chave.

Quando Usar

  • Destacar texto relevante para o usuário
  • Resultados de busca
  • Chamar atenção para algo específico
  • Como um "post-it" digital

Sub e Sup: Subscrito e Sobrescrito

Sub (<sub>) - Subscrito

A tag <sub> cria texto subscrito (abaixo da linha normal). É comumente usado em fórmulas químicas e matemáticas.

Exemplos de Fórmulas Químicas

  • Água: H2O
  • Dioxido de carbono: CO2
  • Glicose: C6H12O6
  • Ácido sulfúrico: H2SO4

Outros Usos

  • Notas de rodapé: Texto1
  • Expressões matemáticas: x1, x2, xn

Sup (<sup>) - Sobrescrito

A tag <sup> cria texto sobrescrito (acima da linha normal). É comumente usado em potências matemáticas e ordinais.

Exemplos de Potências Matemáticas

  • Dois ao cubo: 23 = 8
  • Teorema de Pitágoras: x2 + y2 = z2
  • Área do círculo: πr2
  • Velocidade da luz: 3 × 108 m/s

Outros Usos

  • Ordinais: 1o lugar, 2o lugar, 3o lugar
  • Notas de rodapé: Texto1
  • Referências: Estudo2 mostra que...

Pre (<pre>) - Texto Pré-formatado

A tag <pre> preserva espaços em branco e quebras de linha. O texto é exibido em fonte monoespaçada, exatamente como está escrito.

Quando Usar

  • Código de programação
  • Poemas com formatação específica
  • ASCII art
  • Qualquer texto onde a formatação é importante

Exemplo: Código JavaScript

function exemplo() {
    console.log("Olá, mundo!");
    let x = 10;
    let y = 20;
    return x + y;
}

Exemplo: Poema

    No meio do caminho
        tinha uma pedra
    tinha uma pedra
        no meio do caminho
            tinha uma pedra.

Exemplo: ASCII Art

    /\
   /  \
  /    \
 /______\

Comparação: Com e Sem <pre>

Sem <pre> (espaços colapsados):

Este texto tem muitos espaços e quebras de linha, mas o navegador os ignora.

Com <pre> (formatação preservada):

    Este texto
        tem     muitos     espaços
    e quebras de linha,
        e o navegador preserva tudo!

Combinações de Formatação

Você pode combinar diferentes tags de formatação para criar efeitos mais complexos.

Exemplos de Combinações

Este texto tem ênfase forte combinando <strong> e <em>.

Fórmula química com destaque: H2SO4 (ácido sulfúrico)

Potência matemática importante: x2 + y2 = z2 (Teorema de Pitágoras)

Texto marcado com importância forte combinando <mark> e <strong>.

Nome científico em itálico com nota: Homo sapiens1

Boas Práticas de Formatação

Prefira Tags Semânticas

  • Use <strong> ao invés de <b> quando fizer sentido
  • Use <em> ao invés de <i> para ênfase
  • Tags semânticas são melhores para acessibilidade e SEO

Use Apropriadamente

  • <mark>: Para destacar texto relevante
  • <sub>: Para fórmulas químicas e matemáticas
  • <sup>: Para potências e ordinais
  • <pre>: Quando a formatação é parte do conteúdo

Evite Uso Excessivo

Não use formatação em excesso. Texto com muitas formatações diferentes pode ser difícil de ler e perder o impacto visual.
