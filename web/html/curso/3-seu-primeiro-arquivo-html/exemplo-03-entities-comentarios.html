HTML Entities

Caracteres com Significado Especial

Para escrever tags HTML como texto, use entities:

  • <p> - Tag de parágrafo
  • <div> - Tag de divisão
  • <h1> - Tag de título

Para escrever o símbolo & (e comercial), use: &amp;
