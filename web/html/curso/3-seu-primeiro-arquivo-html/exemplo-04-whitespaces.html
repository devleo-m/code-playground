Indentação para Legibilidade

Código Bem Formatado

A indentação ajuda a visualizar a hierarquia do código. Veja como o código HTML está organizado com indentação consistente. 

<html>
    <head>
        <title>Página</title>
    </head>
    <body>
        <h1>Título</h1>
    </body>
</html>

Código Sem Formatação (Difícil de Ler)

<html><head><title>Página</title></head><body><h1>Título</h1></body></html>

Este código funciona, mas é muito difícil de ler e manter!
