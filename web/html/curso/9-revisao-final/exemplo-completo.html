1. Estrutura e Semântica HTML5

Hierarquia de Títulos (Aula 4)

A hierarquia correta de títulos é essencial para SEO e acessibilidade. Use apenas um H1 por página e mantenha a ordem lógica.

Elementos Semânticos (Aula 7)

HTML5 introduziu elementos semânticos que descrevem o significado do conteúdo, não apenas sua aparência. Isso melhora acessibilidade e SEO.

Formatação de Texto (Aula 4, 7)

Podemos usar várias tags para formatar texto:

  • strong - Importância forte (semântico)
  • b - Negrito (visual)
  • em - Ênfase (semântico)
  • i - Itálico (visual)
  • mark - Texto marcado
  • H2O - Subscrito
  • E=mc2 - Sobrescrito

7. Atributos HTML

Atributos Globais (Aula 5)

  • id: Identificador único (ex: id="secao1")
  • class: Classe para estilização (ex: class="destaque")
  • data-*: Dados customizados (ex: data-usuario="123")
  • title: Tooltip ao passar o mouse
  • lang: Idioma do elemento

Este elemento usa atributos data-* para armazenar informações customizadas.
