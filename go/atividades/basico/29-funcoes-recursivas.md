# Atividade 29: Funções Recursivas

## Objetivo
Praticar funções recursivas básicas.

## Enunciado
Crie um programa que:
1. Crie função recursiva `fatorial` que calcula fatorial de n (n! = n * (n-1)!)
2. Crie função recursiva `fibonacci` que retorna o n-ésimo número de Fibonacci
   - F(0) = 0, F(1) = 1, F(n) = F(n-1) + F(n-2)
3. No `main`, calcule fatorial de 5 e Fibonacci de 7

## Exemplo de Saída
```
Fatorial de 5: 120
Fibonacci(7): 13
```

## Dicas
- Recursão: função chama a si mesma
- Sempre tenha caso base (condição de parada)
- Fatorial: `n! = n * (n-1)!` com `0! = 1`
- Fibonacci: `F(n) = F(n-1) + F(n-2)` com `F(0)=0, F(1)=1`
