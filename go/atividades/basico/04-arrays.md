# Atividade 04: Arrays

## Objetivo
Praticar criação e manipulação de arrays em Go.

## Enunciado
Crie um programa que:
1. Declare um array de 5 inteiros com valores [10, 20, 30, 40, 50]
2. Imprima o array completo
3. Imprima o primeiro elemento
4. Imprima o último elemento
5. Modifique o elemento do meio (índice 2) para 100
6. Imprima o array modificado

## Exemplo de Saída
```
Array completo: [10 20 30 40 50]
Primeiro elemento: 10
Último elemento: 50
Array modificado: [10 20 100 40 50]
```

## Dicas
- Array: `[tamanho]tipo{valores}`
- Acesse elementos com `array[indice]`
- Primeiro índice é 0, último é `len(array)-1`
