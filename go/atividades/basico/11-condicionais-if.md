# Atividade 11: Condicionais - if

## Objetivo
Praticar estruturas condicionais básicas com `if`.

## Enunciado
Crie um programa que:
1. Declare uma variável `idade` com valor 20
2. Use `if` para verificar se a pessoa é maior de idade (>= 18)
3. Se for maior, imprima "Maior de idade"
4. Use `if-else` para verificar se pode dirigir (>= 18)
5. Se não puder, imprima "Menor de idade, não pode dirigir"
6. Use `if-else if-else` para classificar a idade:
   - < 13: "Criança"
   - 13-17: "Adolescente"
   - 18-64: "Adulto"
   - >= 65: "Idoso"

## Exemplo de Saída (idade = 20)
```
Maior de idade
Pode dirigir
Classificação: Adulto
```

## Dicas
- `if condicao { }`
- `if condicao { } else { }`
- `if condicao1 { } else if condicao2 { } else { }`
- Não use parênteses na condição
