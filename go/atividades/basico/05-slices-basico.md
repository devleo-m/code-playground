# Atividade 05: Slices Básico

## Objetivo
Praticar criação e manipulação básica de slices.

## Enunciado
Crie um programa que:
1. Crie um slice de strings com valores ["Maçã", "Banana", "Laranja"]
2. Imprima o slice completo
3. Adicione "Morango" ao slice usando `append`
4. Adicione "Uva" e "Abacaxi" ao slice em uma única operação
5. Imprima o slice final e seu tamanho

## Exemplo de Saída
```
Slice inicial: [Maçã Banana Laranja]
Slice após adicionar Morango: [Maçã Banana Laranja Morango]
Slice final: [Maçã Banana Laranja Morango Uva Abacaxi]
Tamanho do slice: 6
```

## Dicas
- Slice: `[]tipo{valores}`
- `append(slice, elemento)` retorna novo slice
- Sempre atribua o resultado do `append`
- Use `len()` para obter o tamanho
