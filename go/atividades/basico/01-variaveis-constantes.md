# Atividade 01: Variáveis e Constantes

## Objetivo
Praticar declaração de variáveis usando `var` e `:=`, e criação de constantes.

## Enunciado
Crie um programa que:
1. Declare uma variável `nome` do tipo `string` usando `var` e atribua seu nome
2. Declare uma variável `idade` usando `:=` e atribua sua idade
3. Declare uma constante `PI` com valor 3.14159
4. Declare uma constante `LINGUAGEM` com valor "Go"
5. Imprima todas as variáveis e constantes

## Exemplo de Saída
```
Nome: João Silva
Idade: 25
PI: 3.14159
Linguagem: Go
```

## Dicas
- Use `var` para declaração explícita
- Use `:=` dentro de funções para declaração curta
- Use `const` para constantes
- Use `fmt.Printf` ou `fmt.Println` para imprimir
