# Atividade 28: Loops Aninhados

## Objetivo
Praticar loops aninhados e padrões.

## Enunciado
Crie um programa que:
1. Use loop aninhado para imprimir uma tabela de multiplicação 5x5
2. Use loop aninhado para imprimir um padrão de asteriscos:
   ```
   *
   **
   ***
   ****
   *****
   ```
3. Use loop aninhado para encontrar e imprimir todos os pares de números de 1 a 10 cuja soma é 10

## Exemplo de Saída
```
Tabela de multiplicação:
1  2  3  4  5
2  4  6  8  10
3  6  9  12 15
4  8  12 16 20
5  10 15 20 25

Padrão de asteriscos:
*
**
***
****
*****

Pares que somam 10:
1 + 9 = 10
2 + 8 = 10
...
```

## Dicas
- Loop aninhado: `for { for { } }`
- Use variáveis diferentes para cada loop
- Use `fmt.Printf` para formatar tabelas
- Use `break` para sair de loops aninhados se necessário
